# Secure Login Authentication System

A professional cybersecurity-focused secure login web application developed using Flask, SQLite, and bcrypt password hashing.

This project demonstrates secure authentication techniques including encrypted password storage, SQL injection prevention, session management, protected dashboard access, and secure user handling practices.

The application is designed with a modern cybersecurity dashboard interface and multiple software modules to simulate a real-world secure authentication platform.

## Features

- User Registration System
- Secure User Login
- bcrypt Password Hashing
- SQL Injection Protection
- Session Management
- Secure Logout Functionality
- Password Validation
- Protected Dashboard Access
- User Profile Management
- Security Monitoring Dashboard
- Cybersecurity Themed Interface
- Responsive Multi-Page Design
- Optional Two-Factor Authentication Module

## Technologies Used

- Python
- Flask
- SQLite
- bcrypt
- HTML
- CSS

## Security Features

### Password Hashing

User passwords are securely encrypted using bcrypt before storing them in the database.

### SQL Injection Prevention

Parameterized SQL queries are used to protect the database from SQL injection attacks.

### Session Management

Secure session handling is implemented to prevent unauthorized dashboard access.

### Authentication Protection

Protected routes ensure only authenticated users can access sensitive pages.
